# FFXIVSimulator
> FFXIV机制模拟器

### 安装依赖
```js
npm install
```

### 运行项目
```js
npm run dev
```

### 打包
```js
npm run build
```
### 包含插件
> `react-router`,`axios`

### 目录结构
```
—————— public 静态文件
｜

｜  ｜

｜  ｜

｜  ｜
```
